## Quick Setup

1. Ensure you have [Node.js](https://nodejs.org/) or [Deno](https://deno.com/) installed. Check your installation by running:

   ***For nodejs***
   ```bash
    node -v
   ```

   ***For deno***
   ```bash
    deno -v
   ```
    

2. In the project directory, install dependencies:

   ***For nodejs***
   ```bash
    node install
   ```

   ***For deno***
   ```bash
    deno install
   ```

3. Start the development server:

   ***For nodejs***
   ```bash
    npm run dev --open
   ```

   ***For deno***
   ```bash
    deno task dev --open
   ```

4. View the app in the browser.
 
5. Customize the template by navigating to the `/src/components` directory.

   The page will reload if you make edits, and you will see any lint errors in the console.
